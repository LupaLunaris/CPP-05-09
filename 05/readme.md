# CPP Module 05 - Repetition and Exceptions

## Description
This module explores repetition and exception handling in C++98.

## Exercises

### Ex00: Bureaucrat
Implementation of a Bureaucrat class with grade management and custom exceptions.

### Ex01: Form and Bureaucrat
Addition of a Form class that can be signed by bureaucrats.

### Ex02: Concrete Forms
Implementation of ShrubberyCreationForm, RobotomyRequestForm, and PresidentialPardonForm.

### Ex03: Intern
Creation of an Intern class capable of dynamically generating forms.

make        # Compile
